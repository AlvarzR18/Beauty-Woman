## 📋 Descripción

**BeautyWoman** es una tienda virtual especializada en accesorios de joyería y bisutería, diseñada para ofrecer una experiencia de compra elegante y moderna. La plataforma permite a los clientes explorar un catálogo de productos, filtrar por precio, y gestionar su carrito de compras de manera intuitiva.

### 🎯 Características Principales

| Característica | Descripción |
|----------------|-------------|
| 🛍️ **Catálogo de productos** | Visualización en grid con imágenes de alta calidad |
| 💰 **Filtro de precios** | Búsqueda personalizada por rango de precio |
| 🛒 **Carrito de compras** | Desplegable, dinámico y fácil de usar |
| 📱 **Modal de producto** | Vista detallada con descripción y acciones |
| 🎨 **Tres estilos visuales** | Dorado, Plateado y Neutral (intercambiables) |
| 📄 **Sistema de pedidos** | Generación automática de PDF con factura |
| 💬 **Integración WhatsApp** | Envío de pedidos directamente al negocio |
| 📱 **Diseño responsivo** | Adaptable a todos los dispositivos |
